Задание 26(повышенной сложности)
Для каждого посетителя с именем PT_FIRST_NAME выведите дату и время первой экскурсии
SELECT DISTINCT ON(starts_on) starts_on
FROM visitor, visitor_schedule, schedule 
WHERE visitor.visitor_id = visitor_schedule.visitor_id AND schedule.schedule_id = visitor_schedule.schedule_id AND visitor.first_name = PT_FIRST_NAME
ORDER  BY starts_on;

Задание 27(повышенной сложности)
Выведите id сеансов на которые PT_STARTS_ON записались больше 15 человек 
SELECT schedule.id
FROM visitor, visitor_schedule, schedule 
WHERE visitor.visitor_id = visitor_schedule.visitor_id AND schedule.schedule_id = visitor_schedule.schedule_id AND schedule.starts_on ::text LIKE 'PT_STARTS_ON%'
GROUP BY schedule.schedule_id
HAVING COUNT(visitor_id) > 15;

Задание 28(средней сложности)
Выведите названия всех экспонатов, год создания которых больше, чем год создания экспоната с exhibit_id = PT_EXHIBIT_ID

Выведите количество посетителей самой популярной (количество посетителей наибольшее) экскурсии PT_STARTS_ON
SELECT MAX(visitor_count) 
FROM( SELECT COUNT(visitor_id) AS visitor_count
           	FROM visitor_schedule, schedule 
WHERE schedule.schedule_id = visitor_schedule.schedule_id AND schedule.starts_on ::text LIKE 'PT_STARTS_ON%' 
GROUP BY schedule_id
       ) visitor_count

Выведите названия залов, где какой-либо из экспонатов сделан в стране PT_COUNTRY_OF_CREATION
SELECT hall_name FROM hall WHERE hall_id = ANY(SELECT hall_id FROM exhibit WHERE country_of_creation = PT_COUNTRY_OF_CREATION)

Выведите названия залов где все экспонаты сделаны в PT_COUNTRY_OF_CREATION

Выведите количество мужчин, которые записались на экскурсии PT_STARTS_ON
SELECT COUNT(visitor_id) 
FROM visitor
JOIN visitor_schedule using (visitor_id)
JOIN schedule using (schedule_id)
WHERE visitor.gender = 'М' AND schedule.starts_on ::text LIKE 'PT_STARTS_ON%'

Выведите общую стоимость продажи билетов в каждом из музеев PT_STARTS_ON
SELECT SUM(price) 
FROM museum 
LEFT JOIN (SELECT price, museum_id FROM excursion 
		   JOIN schedule using (excursion_id)
AND schedule.starts_on ::text LIKE 'PT_STARTS_ON%' ) excursion using (museum_id) 
GROUP BY museum_name
